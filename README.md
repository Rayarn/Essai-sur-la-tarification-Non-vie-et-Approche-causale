## 1 Description de la table CONTRATS

La table **CONTRATS** regroupe les informations contractuelles relatives à un portefeuille
d’assurance automobile composé d’environ **824 000 lignes** (avenants).

Chaque ligne correspond à une période de couverture associée à un véhicule assuré
et à un conducteur donné.

Cette table contient :

- des informations d’identification du sociétaire et du véhicule ;
- des données temporelles (périodes de validité des contrats, informations liées à l’ancienneté du conducteur et du véhicule) ;
- des caractéristiques techniques du véhicule (catégorie, énergie, segment, puissance, valeur, classification SRA, etc.) ;
- des informations contractuelles (formule souscrite, niveau de franchise, options et garanties choisies) ;
- des indicateurs de profil de risque et d’historique assurantiel (bonus-malus, sinistralité antérieure, usage du véhicule) ;
- des variables socio-démographiques décrivant le conducteur (situation familiale, profession, localisation, etc.).

## 2 Description de la table SINISTRES

La table **SINISTRES** regroupe les informations relatives aux sinistres déclarés
sur le portefeuille d’assurance automobile.

Chaque ligne correspond à un sinistre associé à un véhicule assuré et à un sociétaire donné.
La table contient des informations permettant d’identifier le dossier, le risque concerné,
le type d’évènement ainsi que la date de survenance.

Elle inclut également des données de suivi technique et comptable, notamment :

- le statut du sinistre (en cours ou clos) ;
- les montants de charge (CTB) ;
- les montants de recours.

Ces informations sont disponibles selon plusieurs visions temporelles :

- sinistres des exercices 2014, 2015 et 2016 observés à différentes dates d’arrêté ;
- vision actualisée au 31/05/2017 permettant d’analyser le développement des sinistres.
